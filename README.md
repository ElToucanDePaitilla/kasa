Kasa
Description
Kasa est une application de location d'appartements développée dans le cadre du parcours Développeur Front-End d'OpenClassrooms. Ce projet consiste à construire le front-end d'une application de location immobilière en utilisant React et React Router pour créer une expérience utilisateur moderne et réactive. Les utilisateurs peuvent consulter une liste de logements, voir les détails de chaque bien, et explorer des informations spécifiques comme les équipements et les descriptions.

L'application est déployée en ligne et accessible à l'adresse suivante : https://kasa-eltoucandepaitilla.netlify.app/

L'application repose sur les compétences suivantes :

Développement front-end avec React : Création de composants modulaires et gestion de la logique de présentation.
Routage avec React Router : Configuration et gestion de la navigation entre les pages pour une expérience utilisateur fluide.
Gestion de données simulées : Utilisation de données provenant d'un fichier JSON pour simuler un backend et permettre l'affichage dynamique de contenu.
Utilisation de Create React App : Configuration rapide de l'environnement de développement avec Create React App pour démarrer le projet.
Node.js : Utilisation de Node.js pour exécuter du code JavaScript en dehors du navigateur.
SASS pour la gestion des styles : Structuration des styles CSS de manière optimale avec SASS pour organiser et maintenir le code CSS.
Animations CSS : Intégration d'animations pour améliorer l'expérience utilisateur.
React.StrictMode : Activation de React.StrictMode pour détecter les problèmes potentiels et encourager les bonnes pratiques de développement.
ESLint : Configuration d’ESLint pour assurer la qualité et la cohérence du code en respectant des standards élevés.
Ce projet permet de se familiariser avec les technologies modernes de développement front-end et de se perfectionner dans la création d’interfaces utilisateur dynamiques et attrayantes, très demandées dans l'industrie du web.

npm run eject
Remarque : cette opération est irréversible. Une fois que vous avez utilisé eject, vous ne pouvez plus revenir en arrière !

Si vous n'êtes pas satisfait des choix de configuration et d'outils, vous pouvez à tout moment exécuter eject. Cette commande supprimera la dépendance de build unique de votre projet.

Elle copiera plutôt tous les fichiers de configuration et les dépendances transitives (webpack, Babel, ESLint, etc.) directement dans votre projet, vous donnant un contrôle total sur la configuration. Toutes les commandes, sauf eject, continueront à fonctionner, mais pointeront vers les scripts copiés, que vous pourrez personnaliser. À ce stade, vous êtes seul responsable de la maintenance de cette configuration.

Vous n'êtes jamais obligé d'utiliser eject. La configuration proposée par défaut est adaptée pour des déploiements de taille petite à moyenne, mais si vous souhaitez davantage de personnalisation, cette commande reste disponible.

Kasa
Description
Kasa is an apartment rental application developed as part of the Front-End Developer path at OpenClassrooms. This project involves building the front-end of a real estate rental application using React and React Router to create a modern and responsive user experience. Users can view a list of available properties, see the details of each property, and explore specific information such as amenities and descriptions.

The application is deployed online and accessible at the following address: https://kasa-eltoucandepaitilla.netlify.app/

The application is built upon the following skills:

Front-end development with React: Creation of modular components and management of presentation logic.
Routing with React Router: Configuration and management of navigation between pages for a smooth user experience.
Handling mock data: Using data from a JSON file to simulate a backend and enable dynamic content display.
Using Create React App: Quick setup of the development environment with Create React App to start the project.
Node.js: Using Node.js to execute JavaScript code outside of the browser.
CSS management with SASS: Structuring CSS styles optimally with SASS to organize and maintain CSS code.
CSS animations: Integration of animations to enhance the user experience.
React.StrictMode: Activating React.StrictMode to detect potential issues and encourage good development practices.
ESLint: Configuring ESLint to ensure code quality and consistency by adhering to high standards.
This project provides hands-on experience with modern front-end development technologies and advances proficiency in creating dynamic and visually appealing user interfaces, highly sought after in the web development industry.

You don’t have to ever use eject. The curated feature set is suitable for small and middle deployments, and you shouldn’t feel obligated to use this feature. However, we understand that this tool wouldn’t be useful if you couldn’t customize it when you are ready.

Kasa
Descripción
Kasa es una aplicación de alquiler de apartamentos desarrollada como parte del curso Desarrollador Front-End de OpenClassrooms. Este proyecto consiste en construir el front-end de una aplicación de alquiler inmobiliario utilizando React y React Router para crear una experiencia de usuario moderna y reactiva. Los usuarios pueden ver una lista de propiedades disponibles, consultar los detalles de cada propiedad y explorar información específica como las comodidades y las descripciones.

La aplicación está desplegada en línea y es accesible en la siguiente dirección: https://kasa-eltoucandepaitilla.netlify.app/

La aplicación se basa en las siguientes competencias:

Desarrollo front-end con React: Creación de componentes modulares y gestión de la lógica de presentación.
Enrutamiento con React Router: Configuración y gestión de la navegación entre las páginas para una experiencia de usuario fluida.
Gestión de datos simulados: Uso de datos de un archivo JSON para simular un backend y permitir la visualización dinámica de contenido.
Uso de Create React App: Configuración rápida del entorno de desarrollo con Create React App para iniciar el proyecto.
Node.js: Uso de Node.js para ejecutar código JavaScript fuera del navegador.
Gestión de estilos con SASS: Estructuración óptima de los estilos CSS con SASS para organizar y mantener el código CSS.
Animaciones CSS: Integración de animaciones para mejorar la experiencia de usuario.
React.StrictMode: Activación de React.StrictMode para detectar posibles problemas y fomentar buenas prácticas de desarrollo.
ESLint: Configuración de ESLint para asegurar la calidad y consistencia del código siguiendo altos estándares.
Este proyecto permite familiarizarse con tecnologías modernas de desarrollo front-end y perfeccionar las habilidades en la creación de interfaces de usuario dinámicas y visualmente atractivas, muy demandadas en la industria del desarrollo web.
